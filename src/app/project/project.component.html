Computer Simulation - presents the transmission of an 8-bit word, secured by the Hamming code. The user decides about:
- Transmission medium (ethernet, WI-FI, satellite)
- Error Probability
- Min and Max time for operations performed during simulation
- number of iterations

Technologies and tools:
Angular 6, TypeScript, CSS3, HTML 5, Chart.js, PS CS6

Version control: GitLab

Hybrid crossplatform mobile application.

Technologies and tools:
Ionic 3 (Angular 4/Scss/HTML), plugins for Ionic, Firebase services (DB/host/auth), PS CS6, IonicPro(Version control/Feedback/Monitor)
